# 2-4. Name Cases Assignment

## Code Screenshot

```python
# Variable representing a person's name in different cases
name = "arthur"

# Display the name in different cases
print("Original variable:", name)
print("Lowercase:", name.lower())
print("Uppercase:", name.upper())
print("Title case:", name.title())
```

## Output Screenshot

```
Original variable: arthur
Lowercase: arthur
Uppercase: ARTHUR
Title case: Arthur
```

## Explanation

This Python program demonstrates how to manipulate a string variable representing a person's name using Python's built-in string methods:

1. **Original variable**: Stores "arthur" in the variable `name`
2. **Lowercase**: Uses `.lower()` method to display the name in all lowercase
3. **Uppercase**: Uses `.upper()` method to display the name in all uppercase
4. **Title case**: Uses `.title()` method to display the name with the first letter capitalized

These string methods are useful for formatting names or text in different ways without modifying the original variable.
